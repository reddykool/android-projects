Features :
===========================================================================
Pictures -> shown in Grid view(150wx200h) 
         -> each grid item(pic) open to fullscreen detail view(with only navigation bar)
         -> Detail view -> supports left/right swipe for next/prev item

Videos  -> shows corresponding grid thumbnail (150wx150h)
        -> open to play with Media controls in seperate full activity mode.

Music   -> Rhymes/songs/lullabies show list of corresponding music files and play on click

Main list -> Custom ListView item with category-Text and category-Icon

App theme -> Color Orange (primary bg)
             Color Purple (action bar, status bar)
